\# S-10 Window Sticker Generator (1982–2004)



A static, offline-capable web application that generates reconstructed

Monroney-style window stickers for Chevrolet S-10 trucks from 1982–2004.



Users can enter a VIN (and optionally RPO codes) to produce a data-driven

sticker based on curated S-10 model-year tables, trim/engine mappings,

and pricing references.



The app runs locally or online using only HTML, CSS, JavaScript, and JSON.

No backend is required.



\## Features

\- VIN validation and decoding (1982–2004)

\- Optional RPO code input for more accurate configuration

\- Monroney-style sticker rendering

\- Fully static architecture (local or cloud)

\- Multi-AI collaboration workflow
