\# AI Collaboration Rules



\## Purpose

Multiple AI systems will contribute to this project. These rules ensure

consistency, safety, and schema integrity.



\## Rules

1\. Do not modify files outside your assigned scope.

2\. Follow the schemas in `/docs/data\_model.md` exactly.

3\. Output only valid JSON when editing data files.

4\. Do not invent new fields or structures.

5\. Do not overwrite unrelated sections of code.

6\. Do not change naming conventions.

7\. When unsure, ask for clarification rather than guessing.

8\. All generated files must remain static and browser-compatible.

9\. No backend code is allowed.

10\. All pricing must be marked as approximate unless sourced.



\## File Ownership

\- Claude: All JSON data files

\- Copilot: JS modules

\- GPT: HTML/CSS

\- DeepSeek: Logic refinement

\- Gemini: UI/UX improvements

\- Kimi: Text content

\- LeChat: Review

\- Pi: Simplification
